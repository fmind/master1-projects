STRUCTURE DU DOSSIER
    - *.xmi: versions des diagrammes au format XMI
    - diagrammes: export graphique des diagrammes de la version finale
    - docs: source du rapport au format LYX et captures d'écran du prototype
    - Monopoly: source du projet avec Netbeans
    - prototype: exécutable du prototype
    - versions: versions sous pour ArgoUML


EN CAS D'ERREUR LORS DE L'OUVERTURE DES FICHIERS XMI
    Ma version d'ArgoUML ne réussit pas à importer les fichiers XMI après export
    mais les fichers contiennent toutes les données. Vous pouvez toujours 
    consulter les export au format graphique des diagrammes dans le dossier
    "diagrammes" ou consulter les sources dans le dossier "versions".
    En cas de problème, merci de me contacter.


UTILISATION DU PROTOTYPE
    Pour lancer le prototype, rendez-vous dans le dossier "prototype" en ligne
    de commande et tapez: "java -jar Monopoly.jar".
    Vous pouvez consulter le détail de l'utilisation dans la section 3.2 du
    rapport de projet.
